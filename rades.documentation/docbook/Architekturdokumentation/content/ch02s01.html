Technische Randbedingungen

Programmiersprache

Für die Entwicklung wird als Programmiersprache Java eingesetzt.

Versionsverwaltung

Als Versionsverwaltung kommt Git zum Einsatz. Als Hoster eignet sich Github. Über Github lassen sich auch die Bugtracking und Projekt Homepage realisieren.

Build und Deployment

Für Build- und Deployment prozesse soll initial mit Maven 3 gestartet werden. Später soll ein anderes Werkzeug zum Einsatz kommen.

Continious Integration

Für die Realisierung des CI Prozesses soll Jenkins in Form eines SaaS zum Einsatz kommen. Als Hoster eignet sich CloudBees.
